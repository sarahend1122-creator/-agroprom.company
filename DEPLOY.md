# Деплой Agroprom на Vercel

## Ошибка: «Не удалось найти каталоги `pages` или `app`»

Она появляется, когда Vercel собирает проект **не из той папки**, где лежат `app/` и `package.json`.  
В коде ничего менять не нужно — только настройка в Vercel.

---

### Что сделать по шагам

1. **GitHub**  
   Открой свой репозиторий в браузере. Посмотри **корень** (самый верхний уровень):
   - Видишь **одну папку** (например `rr`, `agroprom`, `GiftHab`), а внутри неё — `app`, `package.json`?  
     → Запомни имя этой папки (например `rr`).
   - Видишь сразу папку `app` и файл `package.json` в корне?  
     → Root Directory в Vercel оставь **пустым**.

2. **Vercel**  
   Зайди на vercel.com → выбери проект → вкладка **Settings**.

3. В блоке **General** найди **Root Directory** → нажми **Edit**.

4. Введи **только имя папки** из шага 1 (например `rr`), без слэшей.  
   Если проект в корне репозитория — оставь поле пустым.  
   Нажми **Save**.

5. Вкладка **Deployments** → у последнего деплоя **⋯** → **Redeploy**.

После этого сборка должна идти из правильной папки и ошибка пропадёт.

---

В корне проекта есть файл **`VERCEL_НАСТРОЙКА.txt`** — там та же инструкция кратко.
